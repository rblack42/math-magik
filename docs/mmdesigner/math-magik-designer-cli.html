Math Magik Designer CLI¶

The Math-Magik design application, named mmd, can be launched as either a command line tool, or as a graphical user interface tool.

Note

The GUI version is undergoing development at present.

Command Line Interface¶

In this section, we will show the mmd application can be used in your model design work.

First, just typing mmd on the command line will display a help message to show you all available commands and options that can be used.

System Message: ERROR/6 (/Users/rblack/_dev/math-magik/rst/mmdesigner/math-magik-designer-cli.rst, line 112)

Command ['mmd'] failed: [Errno 2] No such file or directory: 'mmd'

You can also ask for help explicitly”

System Message: ERROR/6 (/Users/rblack/_dev/math-magik/rst/mmdesigner/math-magik-designer-cli.rst, line 116)

Command ['mmd', '--help'] failed: [Errno 2] No such file or directory: 'mmd'

Unfortunately, this help message does not show you everything about possible commands. The list of commands shown have additional parameters you can see by asking for help on them:

System Message: ERROR/6 (/Users/rblack/_dev/math-magik/rst/mmdesigner/math-magik-designer-cli.rst, line 122)

Command ['mmd', 'stl', '--help'] failed: [Errno 2] No such file or directory: 'mmd'

Here the -all parameter will attempt to do a STL files for the entire design. (This can take a while!)
