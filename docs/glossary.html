Glossary¶

Path¶

The exact location of any file or folder in a file system is defined by specifying the “path” to that resource. The path begins with the Root Directory, followed by the names of each subfolder leading to the resource. Between each folder name, we place the Path Separator character.

Path Separator¶

The character used to separate directory names in a path. On Linux and Mac this separator character is a forward slash. On Windows systems, it is a backward slash.

Root Directory¶

The top lever directory on your file system. On Windows systems, this is usually C:\
