Math Magik LPP¶

This is the design documentation for the Math Magik LPP indoor model airplane. Browse this section to see details about how this design is organized, and to see the actual code generated.

System Requirements¶

If you want to run the code in this project, you need to use a system with these tools installed:

  • Python version 3+

  • OpenSCAD

Both of these are simple to install on most modern computer systems. I use a Macbook Pro laptop to do my development work.
