Glossary
########

..  glossary::
    :sorted:

    Path
        The exact location of any file or folder in a file
        system is defined by specifying the "path" to that resource. The path
        begins with the :term:`Root Directory`, followed by the names of each
        subfolder leading to the resource. Between each folder name, we place
        the :term:`Path Separator` character.

    Root Directory
        The top lever directory on your file system. On Windows systems, this
        is usually **C:\\**

    Path Separator
        The character used to separate directory names in a path. On Linux and
        Mac this separator character is a forward slash. On Windows systems, it
        is a backward slash.
