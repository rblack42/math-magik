math-magik-lpp
##############

This is the top level file, which creates a complete Limited Penneyplane model.

The model consists of a collection of assemblies, each of which represents a separable part of the finished design. Individual assemblies are created to facilitate designing storage for the model. An example of this storage is included in the project (see :ref:`flight_box> <flight_box>`).
